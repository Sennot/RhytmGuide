# Rhythm Guide

Minimal rhythm-game style input guide for Geometry Dash.

The mod reads `.gdr` / `.gdr.json` macro inputs and draws clean timing lines on top of gameplay.
When you press, it shows whether you were `In` or how many milliseconds early/late you were.

Main features:

- translucent non-blocking overlay lines
- input deviation feedback in milliseconds
- optional `In` text for good inputs
- ship/wave auto-hide
- CBF-friendly sub-frame input timing
- many visual and timing settings
